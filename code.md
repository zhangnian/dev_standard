# 编码规范

### 带来的好处

- 规范的代码可以提高团队合作效率
- 规范的代码可以减少bug

- 规范的代码可以降低维护成本，提高代码可读性

- 规范的代码有助于代码审查
- 规范的代码有助于程序员自身的成长
